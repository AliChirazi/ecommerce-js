# 🛒 Ecommerce JS

Aplicación de carrito de compras desarrollada con JavaScript puro.

## 🚀 Funcionalidades
- Agregar productos al carrito
- Eliminar productos
- Modificar cantidades (+ y -)
- Persistencia con localStorage
- Cálculo de total dinámico

## 🧠 Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## 🎯 Objetivo
Este proyecto fue desarrollado como práctica para mejorar el manejo del DOM, eventos y lógica de programación en JavaScript.
